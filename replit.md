# Mart John F. Labaco — Personal Portfolio

## Overview
A personal portfolio website for Mart John F. Labaco (JM / Mart), a 17-year-old developer from the Philippines. Built with plain HTML, CSS, and JavaScript, served by a lightweight Node.js HTTP server.

## Stack
- **Runtime**: Node.js (built-in `http` module, no dependencies)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Start command**: `npm start`
- **Port**: 5000

## Project Structure
```
├── server.js           Node.js static file server
├── package.json
└── public/
    ├── index.html      Single-page portfolio
    ├── style.css       All styles (dark navy theme)
    ├── script.js       Particles, typewriter, scroll reveal, skill bars
    └── profile.jpg     Profile photo (Mart's photo)
```

## Features
- Floating particle canvas background with mouse interaction
- Glitch/glow effect on hero name
- Typewriter animation cycling through phrases
- Scroll-reveal animations on every section
- Animated skill progress bars
- Glassmorphism cards
- Fully responsive (mobile + desktop)
- Smooth scroll navbar
- Sections: Hero, About, Skills, Hobbies, Experience, Contact

## Design
- Color scheme: Dark Navy (#050d1a) + Blue accent (#4f8ef7) + Purple (#7c5cbf)
- Font: Outfit (Google Fonts)
- Clean masculine aesthetic with glassmorphism cards
